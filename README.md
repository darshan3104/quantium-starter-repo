# Quantium Data Analytics Virtual Experience — Starter Repo

A complete data engineering and visualization project analyzing **Pink Morsel** sales data, built as part of the Quantium Data Analytics program.

---

## 📁 Project Structure

```
quantium-starter-repo/
├── data/                   # Raw CSV source files (sales data by region)
├── tests/
│   └── test_app.py         # Pytest test suite for the Dash app
├── app.py                  # Interactive Dash visualization dashboard
├── process_data.py         # Data processing & transformation script
├── conftest.py             # Pytest configuration (auto-installs ChromeDriver)
├── run_tests.sh            # Bash script to run the full test suite
├── output.csv              # Processed output data (generated by process_data.py)
└── requirements.txt        # Python dependencies
```

---

## 🔧 Setup

### 1. Clone the repository
```bash
git clone https://github.com/darshan3104/quantium-starter-repo.git
cd quantium-starter-repo
```

### 2. Create and activate a virtual environment
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
pip install "dash[testing]" webdriver-manager
```

---

## 🚀 Usage

### Step 1 — Process the raw data
Reads all CSV files from `data/`, filters for **Pink Morsel**, computes sales, and writes `output.csv`.

```bash
python process_data.py
```

**Output columns:** `sales`, `date`, `region`

---

### Step 2 — Run the dashboard
```bash
python app.py
```

Open your browser at **http://127.0.0.1:8050/**

#### Dashboard features:
- 📈 Daily sales line chart
- 📉 7-day rolling average overlay
- 🔴 Vertical marker at Jan 15, 2021 (price increase event)
- 🟢/🟠 Horizontal lines for before/after average sales
- 🗂️ Region filter (All, North, East, South, West)
- ✅ Dynamic conclusion badge showing whether sales increased or decreased

---

## 🧪 Running Tests

### Using the bash script (recommended)
```bash
bash run_tests.sh
```

### Using pytest directly
```bash
pytest tests/test_app.py -v
```

### What the tests verify:
| Test | Description |
|---|---|
| `test_header_present` | Page title renders with correct text |
| `test_graph_present` | Plotly chart (`#sales-graph`) is rendered |
| `test_region_picker_present` | Radio filter (`#region-filter`) has 5 options |
| `test_region_labels_correct` | Labels match All / North / East / South / West |

> **Note:** Tests use `dash[testing]` + Selenium. `conftest.py` automatically downloads the correct ChromeDriver via `webdriver-manager` — no manual setup required.

## 📊 Business Insight

The dashboard answers: **Did the Jan 15, 2021 price increase help or hurt sales?**

The conclusion is displayed dynamically on the dashboard based on the filtered region's data.
